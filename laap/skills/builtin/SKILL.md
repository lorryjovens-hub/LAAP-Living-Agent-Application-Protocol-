# LAAP Helper

## Overview
Check the LAAP agent's status, memory usage, and cognitive state.

## Usage
Just type `/laap-helper` to invoke this skill.

The agent will:
1. Check current status
2. Report memory statistics
3. Show cognitive state (needs, emotions, goals)
4. Provide helpful suggestions
